Regular Expressions (Regex) Study

In this study, you'll be shown a description of a regex pattern in English. This description was given to an AI language model to generate candidate regexes that might match that pattern.

You will be shown words related to the pattern description and regexes. Classify the words based on whether they match that description exactly. The words will be shown in pairs, but each word is independent. Both, neither, or just one could match the description.

Be sure to match the provided specification, and NOT your intuitive understanding of the description.

For example, if the description specifies phone numbers in the format ###-###-####, you should NOT accept words such as:

  • (123)456-7890
  • 123-4567
  • +1 123-456-7890

because, even if these are common ways of providing phone numbers, they do not match what the description specifies.

However, words like 123-456-7890 and 000-001-0101 ARE valid with this description.
